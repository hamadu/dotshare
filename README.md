DOTSHARE
======================
URL: http://www.hama-du.net/dotshare/
## 

ブラウザ上で動くドット絵エディタです。
SHAREなんて仰々しい名前がついていますが現在ドット絵のシェア機能はありません。できればそのうち。

機能追加のためぼちぼち開発中です。

 
ショートカットキー
------
* Ctrl+z  取り消し
* Ctrl+x  やり直し
* z       ズームアウト
* x       ズームイン

 
使える環境
------
Google Chrome(推奨) もしくは Firefox
